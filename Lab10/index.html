XML and JavaScript Overview

What is XML?

XML (eXtensible Markup Language) is a markup language that defines a set of rules for encoding documents in a format which is both human-readable and machine-readable.

How is XML used?

  • Web Development: To transport and store data.
  • Configuration Files: For software settings.
  • Office Applications: For Microsoft Office file formats.

What does XML look like?

<note>
    <to>Reader</to>
    <from>Author</from>
    <heading>Reminder</heading>
    <body>Don't forget to check the example.</body>
</note>

What is JavaScript?

JavaScript is a programming language that is one of the core technologies of the web, alongside HTML and CSS, which can manipulate HTML and CSS to enhance the interactivity of websites.

How is JavaScript related to HTML and CSS?

JavaScript is used in web development to manipulate both HTML and CSS, adding interactive behavior to web pages. It operates on the client side and can interact with HTML elements to modify appearance, content, and behavior.

Common JavaScript Examples

Elements

Accessing an HTML element: document.getElementById('example')

Properties

Changing the style: document.getElementById('example').style.color = 'red';

Methods

Adding an event listener: document.addEventListener('click', functionName);

Functions

A simple function: function greet() { alert('Hello, world!'); }
