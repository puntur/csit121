Exploring Form Elements

An educational guide on form elements and their best practices.

Action

The action attribute specifies where to send the form-data when a form is submitted.


Autocomplete

The autocomplete attribute specifies whether a form or input field should have autocomplete enabled.


ID

The id attribute specifies a unique id for an HTML element. It is often used to target elements with CSS or JavaScript.


Method

The method attribute specifies the HTTP method (GET or POST) to be used when submitting the form data.


Name

The name attribute specifies the name of the form element. It is used to identify the form data after it is submitted.


Enctype

The enctype attribute specifies how the form-data should be encoded when submitting it to the server.
